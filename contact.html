Arquitectura

Nuestra Arquitectura

Usamos

  • Route 53 lo utilizamos para la gestión de DNS y direccionamiento de dominios. Configuramos el dominio para apuntar a la distribucion de CloudFront y posterior al bucket de S3.

  • CloudFront como una red de entrega de contenido para distribuir tu sitio web globalmente y mejorar la velocidad de carga al servir el contenido desde ubicaciones más cercanas a los usuarios finales.

  • AWS Certificate Manager para obtener certificados SSL/TLS gratuitos y configurar HTTPS en tu distribución de CloudFront para asegurar las comunicaciones

  • AWS WAF fue agregado para propocionar una capa de seguridad adicional, puedes utilizar AWS WAF para proteger tu aplicación web de amenazas comunes.
